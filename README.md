# Emotion Recognition (FER2013)

Projet CNN pour reconnaître les émotions humaines depuis une image ou la webcam.

## Fonctionnalités
- Entraînement sur dataset FER2013
- Prédiction sur image
- Détection d'émotion en temps réel (Webcam)
- Version Google Colab + Version PC
